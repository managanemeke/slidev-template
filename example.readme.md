# title

## Initialize and install

```sh
npx degit --force github:managanemeke/slidev-template .
```

```sh
pnpm install
```

```sh
cp example.editorconfig .editorconfig
```

```sh
cp example.gitignore .gitignore
```

```sh
cp example.readme.md readme.md
```

```sh
cp example.slides.md slides.md
```

```sh
cp example.style.css style.css
```

## Run for developing

```sh
pnpm run dev
```

## Export as pdf

```sh
pnpm exec playwright install
```

```sh
pnpm run export
```
